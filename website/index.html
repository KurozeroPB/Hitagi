Support Server

  • h.about Some information about the bot.
  • h.asw Gets a random comic from asofterworld.
  • h.ban Permanently ban someone from the server.
  • h.blacklist Blacklists a member.
  • h.clear Deletes x amount of messages.
  • h.eval
  • h.game Set the "Playing" status.
  • h.guildinfo Some information about the guild.
  • h.invite Sends bot invite and invite to the Support-server.
  • h.kick Temporarily Kick somebody from the server.
  • h.nedr Sends a random comic from nedroid.
  • h.owner Find the owner of a shared server.
  • h.ping Sends pong message with ping delay.
  • h.prune Prune the messages from the bot.
  • h.restart Restarts and updates bot.
  • h.roles Get the guild roles.
  • h.say Make the bot say something.
  • h.status Set the bot as online, idle, dnd or invisible status.
  • h.whitelist Whitelists chosen member.
  • h.whois Displays info about a user.
  • h.xkcd Sends a random comic from xkcd.
